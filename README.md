# 📊 Borsa & Portföy Takip Uygulaması (Android)

Bu proje, Android Studio kullanılarak geliştirilen bir **borsa / portföy takip uygulamasıdır**.  
Amaç; kullanıcının bakiyesini, portföyünü ve kâr/zarar durumunu takip edebilmesini sağlamaktır.

> ⚠️ Proje **henüz tamamlanmamıştır**. Geliştirme süreci devam etmektedir.

---

## 🚀 Özellikler

- 🔐 Firebase Authentication ile kullanıcı girişi
- 💰 Kullanıcı bakiyesi görüntüleme
- ➕ Para yatırma / ➖ para çekme
- 📈 Portföy (hisse) görüntüleme
- 📊 Kâr / zarar hesaplama mantığı
- 🧾 İşlem geçmişi altyapısı
- 👤 Profil ekranı
- 🖼️ Profil fotoğrafı (local storage üzerinden saklanıyor)

---

## 🧠 Kullanılan Teknolojiler

- **Java** (Android)
- **Android Studio**
- **Firebase Authentication**
- **Firebase Firestore**
- **Material Design**
- **SharedPreferences** (local veri saklama)
- **ConstraintLayout & LinearLayout**
